LogiX, the heartbeat of Neos

Neos has a visual scripting system that allowss its users to read and manipulate data in realtime within VR. No need for compiling, no need for processing. No need for programs to break or not run due to errors, if you have an error in your program you see it immediately.

Where to learn

There is a wiki for logix, which can be found here

Currently the best way to learn is to take a look at other user's projects and see different methods for accomplishing goals. The users of NeosVR are always willing to help whenever they can.
